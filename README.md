# FortiSIEM Incident Analysis Assistant

This is a lightweight, analyst-assisted investigation tool designed to help
Tier-1 SOC analysts perform first-response analysis for FortiSIEM incidents.

## Key Characteristics
- Manual input (copy-paste from FortiSIEM)
- AI-assisted analysis only
- No automation, blocking, or response actions
- No changes to FortiSIEM configuration
- Designed for internal SOC use

## Usage
1. Paste the FortiSIEM incident details
2. Paste the last 10 related raw log events
3. Click "Analyze Incident"
4. Review structured analysis output

## Disclaimer
AI output is advisory only.
Final incident decisions remain the responsibility of the SOC analyst.
